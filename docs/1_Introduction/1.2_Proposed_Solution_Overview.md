# 1.2 Proposed Solution: An Intelligent Switched AFFOPID Control System

To address the complex control challenges outlined previously, this project implements a sophisticated, multi-layered intelligent control architecture inspired by the advanced methodologies presented by Sime, T.L., et al. in their 2024 paper, "Modeling of genetic algorithm tuned adaptive fuzzy fractional order PID speed control of permanent magnet synchronous motor for electric vehicle."

Our solution is not a single controller, but a hybrid, intelligent system designed to leverage the strengths of several advanced control techniques. The core of our proposed system is a **Switched Adaptive Fuzzy Fractional Order PID (AFFOPID) controller**.

## Core Components of the Proposed Solution

The architecture is composed of three fundamental building blocks, each designed to tackle a specific aspect of the control problem:

### 1. The Fractional Order PID (FOPID) Core

At its heart, the system utilizes a **Fractional Order PID (FOPID)** controller. Unlike a conventional PID controller which has three tuning parameters (`Kp`, `Ki`, `Kd`), the FOPID controller introduces two additional degrees of freedom:
*   The fractional order of integration, **lambda (`λ`)**.
*   The fractional order of differentiation, **mu (`µ`)**.

These fractional orders provide a much finer and more flexible tuning capability. The `FOPID` core allows for a more accurate modeling of the system's dynamics and can achieve a superior balance between `transient response` and `steady-state` performance compared to its integer-order counterpart.

To combat the issues of non-linearity and parameter variations, we implemented a dual-stage **Adaptive Fuzzy Logic** layer on top of the FOPID core. This layer consists of two interconnected Fuzzy Logic Controllers (FLCs) that work in tandem to dynamically tune the FOPID gains (`Kp` and `Ki`) in real-time.

*   **Fuzzy Logic Controller 1 (FLC1):** Acts as the primary gain scheduler. It continuously monitors the system's `Error (E)` and `Change in Error (dE/dt)` and computes the appropriate base values for `Kp` and `Ki`.
*   **Fuzzy Logic Controller 2 (FLC2):** Acts as a fine-tuning module. It also monitors the system state and produces `Scaling Factors` that modulate the base gains generated by FLC1.

This dual-fuzzy structure allows the controller to adapt its behavior aggressively during large transients and gently during small adjustments, effectively mimicking the decision-making process of an expert human operator.

### 3. The Intelligent Switching Logic

A key innovation in our implementation is the **intelligent switching mechanism**. The highly complex AFFOPID system is computationally intensive and is primarily needed during the `transient state`. To optimize performance and ensure stability in the `steady state`, we implemented a switch that automatically transitions between two control modes based on the magnitude of the speed `Error`.

*   **When `|Error|` is large:** The full **AFFOPID** system is active, leveraging the power of fuzzy logic to achieve rapid and stable convergence.
*   **When `|Error|` is small (below a predefined `Threshold`):** The system switches to a simpler, fixed-gain **FOPID** controller. This controller is specifically tuned for optimal `steady-state` performance, ensuring zero `steady-state error` and rejecting minor disturbances without the risk of `chattering`.

## Automated Tuning via Genetic Algorithm (GA)

Manually tuning the numerous parameters of this complex system (`Kp`, `Ki`, `Kd`, `λ`, `µ`, fuzzy rules, etc.) is practically impossible. Therefore, a key part of our project is the implementation of a **Genetic Algorithm (GA)** to automate this process. The GA systematically explores the vast parameter space, simulating the controller's performance for each candidate solution and evolving towards the optimal set of parameters that minimizes a predefined cost function (the Integral of Squared Error, `ISE`).

This comprehensive, multi-faceted approach provides a robust and high-performance solution tailored to the specific challenges of PMSM speed control in electric vehicles.
